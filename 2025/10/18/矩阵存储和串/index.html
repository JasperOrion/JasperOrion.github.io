一、矩阵的压缩存储

需要考虑到矩阵中元素的存取，矩阵的长度，以及存储的方式。

1.数组存储结构

  • 一维数组：数组名为起始地址 LOC，也可看作指向第一个元素的指针；数组元素大小相同，在物理上连续存储，数组元素array[i] 的存放地址为 LOC + i * sizeof (ElemType)。
  • 二维数组：逻辑上二维数组即为的矩阵，下标从0开始，内存中则是需要连续存储在一串连续空间中，先按行存储（数组元素 array [ i ] [ j ] 的存放地址为 LOC + (i * N + j) * sizeof (ElemType)）或先按列进行存储（数组元素 array [ i ] [ j ] 的存放地址为 LOC + (j * M + i) * sizeof (ElemType)）。

2.对称矩阵

  • 是一个n阶方阵，且其中任意一个元素 ai j 都有 ai j = aj i ，关于主对角线对称。

可以只存储主对角线 + 下三角区域 | | 只存储主对角线 + 上三角区域

  • 长度：(1+n)n/2
  • 一维数组下标映射：
    • 条件：1.行优先
      • k = (i - 1) i / 2 + j - 1 (i>=j)
      • k = (j - 1) j / 2 + i - 1 (i < j)
    • 条件：2.列优先
      • k = ( n + n - j + 2 )( j - 1 ) + i - j (i>=j)
      • k = ( n + n - i + 2 )( j - 1 ) + j - i (i < j)

3.上/下三角矩阵

需要额外存储上（下）三角区域的常量，将其映射到最后一个元素

  • 长度：(1+n)n/2 + 1
  • 一维数组下标映射：
    • 下三角矩阵
      • 条件：1.行优先
        • k = (i - 1) i / 2 + j - 1 (i>=j)
        • k = ( n + 1 ) n / 2 (i < j)
      • 条件：2.列优先
        • k = ( n + n - j + 2 )( j - 1 ) + i - j (i>=j)
        • k = ( n + 1 ) n / 2 (i < j)
    • 上三角矩阵
      • 条件：1.行优先
        • k = (i - 1) ( n + n - j + 2 ) / 2 + ( j - i ) (i<=j)
        • k = n ( n + 1) / 2 (i > j)
      • 条件：2.列优先
        • k = j ( j - 1 ) / 2 + i (i<=j)
        • k = n ( n + 1) / 2 (i > j)

4.三对角矩阵

当|i - j| > 1时，有ai j = 0 (1 <= i , j <= n)

  • 长度：3n - 2

按照行优先原则，ai j是第几个元素？

ai j前i - 1 行有 3 ( i - 1 ) - 1个元素 ，ai j第i 行第 j-i+2个元素。k = 2i + j - 2.

按照行优先原则，b[k]是ai j？

即第k+1个元素，位于第i-1行后和第 i 行内3(i-1)<k+1<=3i-1=> i >= (k+2)/3上取整

j = k - 2i + 3.
