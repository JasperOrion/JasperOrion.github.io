Python 3.12 + OpenCV + NumPy + Matplotlib

一、形态学概述

1.1 核心思想

“用结构元素探测图像，提取与结构元素相匹配的形状信息”

数学形态学基于集合论，关注的是图像的几何结构和形态特征。

对比 图像增强 形态学处理
关注点 像素灰度值 目标几何形状
典型操作 拉伸、滤波 膨胀、腐蚀、开/闭
理论基础 灰度变换、卷积 集合论

1.2 基本术语

术语 含义
结构元素 (SE) 小的二值模板（3×3方形/十字/椭圆），中心为锚点
膨胀 扩大白色区域，缩小黑色孔洞
腐蚀 缩小白色区域，扩大黑色孔洞
开运算 先腐蚀后膨胀 → 去噪点、断开窄连接
闭运算 先膨胀后腐蚀 → 填孔洞、连接断口

二、二值形态学

2.1 膨胀与腐蚀 — 手写实现

2.2 膨胀 (Dilation)

$$A \oplus B = {z \mid (\hat{B})_z \cap A \neq \varnothing}$$

  • SE 滑动中，只要与 A 有至少一个重叠点 → 输出白
  • 效果： 白区扩大一圈，黑孔洞缩小/填平，相邻物体连通

2.3 腐蚀 (Erosion)

$$A \ominus B = {z \mid (B)_z \subseteq A}$$

  • SE 必须完全包含在 A 内 → 输出白
  • 效果： 白区缩小一圈，黑孔洞扩大，孤立白点消失

迭代效果： iterations=1/2/3 → 效果倍数叠加。

2.4 开运算与闭运算

$$A \circ B = (A \ominus B) \oplus B$$

  • ✅ 去白噪点 ✅ 断开窄连接 ✅ 平滑轮廓

$$A \bullet B = (A \oplus B) \ominus B$$

  • ✅ 填黑孔洞 ✅ 连接断口 ✅ 平滑轮廓

直观理解：

  • 开运算： 腐蚀把小噪点消掉→膨胀恢复目标大小，但噪点回不来了
  • 闭运算： 膨胀把黑孔填平→腐蚀恢复目标大小，但黑孔回不来了

组合技巧： 先开运算去噪→再闭运算填孔 = 干净连通目标。

2.5 形态学梯度

$$\text{Gradient}(A) = (A \oplus B) - (A \ominus B)$$

运算 顺序 去白噪 填黑孔 提取边缘
开运算 腐蚀→膨胀 ✅ ❌ ❌
闭运算 膨胀→腐蚀 ❌ ✅ ❌
梯度 膨胀−腐蚀 ❌ ❌ ✅

三、灰度形态学

核心区别： 二值用集合交/并，灰度用取最大值/最小值。

3.1 灰度膨胀与腐蚀

$$(f \oplus b)(x,y) = \max_{(s,t)\in b} {f(x+s, y+t)}$$
$$(f \ominus b)(x,y) = \min_{(s,t)\in b} {f(x+s, y+t)}$$

操作 效果 应用
灰度膨胀 取最大→变亮，亮区扩张 去暗细节
灰度腐蚀 取最小→变暗，暗区扩张 去亮细节

四、总结

核心对比表

操作 二值效果 灰度效果 主要用途
膨胀 白区扩大 变亮，亮区扩张 连接、填孔
腐蚀 白区缩小 变暗，暗区扩张 去噪、分离
开运算 去白噪点 去亮细节 去噪首选
闭运算 填黑孔洞 去暗细节 填孔首选

Top-Hat 提取亮区 亮细节/背景 光照校正
Black-Hat 提取暗区 暗细节 缺陷检测

适用场景速查

问题 方法
二值图有白噪声点 开运算(3×3)
二值图有黑孔洞 闭运算(3×3)
既有白噪点又有黑孔洞 先开→后闭
分离相连物体 腐蚀(迭代)
光照不均 Top-Hat(大核)+直方图均衡
提取微小亮细节 Top-Hat(小核)
提取暗缺陷 Black-Hat

核心原则

  1. SE 决定效果 — 选择形状和尺寸匹配目标
  2. 组合使用效果更好 — 开+闭同时去噪和填孔
  3. 核越大效果越强但细节损失越多
  4. 二值是基础，灰度是推广
