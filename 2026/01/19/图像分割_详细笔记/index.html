图像分割 — 完整教学笔记（含代码）

《数字图像处理》教学笔记 · 第四章
Python 3.12 + OpenCV + NumPy + Matplotlib

一、概述

1.1 什么是图像分割

将图像划分为若干互不相交的区域，每个区域具有一致的视觉特性。

地位： 原始图像 → 分割 → 目标提取 → 特征分析 → 识别理解

1.2 基本概念

概念 说明
区域 (Region) 具有相似特性的连通像素集
边缘 (Edge) 相邻区域之间的灰度突变
前景/背景 目标物体 / 其余部分
过分割 一个目标被分成多块
欠分割 多个目标被合并在一起

两个基本原则： 不连续性（边缘法）和相似性（区域法）。

三、基于区域的分割

3.1 阈值法

$$g(x,y) = \begin{cases} 1 & f(x,y) > T \ 0 & f(x,y) \le T \end{cases}$$

Otsu 大津法（全局阈值）

自动寻找使类间方差最大的最优阈值：

$$T^* = \arg\max_T \omega_0\omega_1(\mu_0 - \mu_1)^2$$

方法 原理 适用
Otsu 类间方差最大 双峰直方图
自适应均值 局部均值-C 光照不均
自适应高斯 局部高斯加权均值-C 光照不均

五、总结

三种方法对比

方法 核心依据 需要先验 复杂度 适用
Canny 边缘 梯度不连续性 ❌ 低 轮廓清晰、尺寸测量
Otsu 阈值 灰度直方图分布 ❌ 低 简单背景、高对比度
区域生长 灰度相似性+连通性 ✅种子点 中 特定目标提取
分水岭 地形学+连通性 ✅标记 中 粘连物体分离
显著性 颜色/纹理对比度 ❌ 低-中 自动目标定位

实际工程流程

选择建议

  1. 场景简单 → Otsu
  2. 需要轮廓 → Canny
  3. 光照不均 → 自适应阈值
  4. 特定目标 → 区域生长
  5. 粘连目标 → 分水岭
  6. 完全自动化 → 显著性检测
