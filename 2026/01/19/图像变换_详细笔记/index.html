《数字图像处理》教学笔记 · 第一章
Python 3.12 + OpenCV + NumPy + Matplotlib

一、图像变换概述

1.1 为什么需要变换

问题 变换的解决
空域卷积 $O(N^2M^2)$ 太慢 卷积定理 → 频域乘积 $O(N\log N)$
噪声/纹理/轮廓混叠 频率分解 → 低通/高通分别处理
改一个像素只影响局部 频域一个分量影响全局
周期性图案难检测 频谱对称亮点对
数据压缩 DCT 能量集中 → JPEG

1.2 四种主要变换

变换 特点 应用
几何变换 改变像素坐标 校正、配准
傅里叶变换 (DFT) 复数，幅度+相位 滤波、频域分析
离散余弦变换 (DCT) 实数，能量极集中 JPEG 压缩
小波变换 (DWT) 多分辨率 JPEG2000、去噪

几何变换中逆向映射得浮点坐标，需”猜”灰度值。

方法 像素数 速度 质量
最近邻 1 最快 锯齿严重
双线性 4 中等 平滑，OpenCV默认
双三次 16 最慢 最平滑

四、傅里叶变换

4.1 输入输出

$$F(u,v) = \sum_{x=0}^{M-1}\sum_{y=0}^{N-1} f(x,y) \cdot e^{-j2\pi(ux/M + vy/N)}$$

输入 (x,y) 输出 (u,v)
含义 像素位置→灰度值 频率→复数（幅度+相位）
数据类型 实数 复数

变换不改变数据量——只是换了视角。h×w 实数 → h×w 复数（每个复数 = 2 个实数，总量相同）。

4.2 幅度谱 vs 相位谱

成分 含义 存储的信息
幅度谱 $ F(u,v) $
相位谱 $\angle F(u,v)$ “在哪里+怎么摆” 形状、边缘位置、整体结构

结论：相位比幅度更重要！ 图像的结构信息存储在相位中。

4.3 频谱能读出什么

频谱特征 图像特征
中心亮度 (DC) 平均灰度
水平截线 垂直纹理/边缘
垂直截线 水平纹理/边缘
径向分布 平滑 vs 锐利
角度分布 纹理方向性
对称亮点对 周期性图案

4.4 DFT 的重要性质

  1. 线性性： DFT(A+B) = DFT(A) + DFT(B)
  2. 可分离性： 2D-DFT = 先行后列（或先列后行）
  3. 平移不变性： 空域平移 → 幅度谱不变，相位变化
  4. 旋转性： 空域旋转 θ → 频域同角度旋转
  5. 卷积定理： 空域卷积 ↔ 频域乘积（★最重要）

特性 DFT DCT

应用 频谱分析、滤波 JPEG 压缩

六、知识体系总览

核心原则

  1. 变换不改变数据量——只是换视角
  2. 幅度=强度，相位=位置——相位对视觉更重要
  3. DCT 能量更集中——JPEG 的基础
  4. 频域操作影响全局——改一个频率分量影响整张图
  5. 卷积定理——频域滤波 $O(N\log N)$ 远优于空域卷积 $O(N^2)$
